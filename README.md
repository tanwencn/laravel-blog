# Laravel Blog

Laravel Admin & Blog System, supporting Laravel 5.5 and 5.6!

## Installation Steps

### Require the Package
```bash
composer require tanwencn/laravel-blog
```

### Run The Installer
```bash
php artisan blog:install
```

### Default an Admin User
    URL:http://youwebsite/admin
    email: admin@admin.com   
    password: admin
